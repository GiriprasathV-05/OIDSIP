# 🚀 How To Run Projects

## Requirements

- XAMPP
- PHP
- MySQL
- Web Browser

---

## Steps

1. Install XAMPP
2. Start Apache & MySQL
3. Move project folder into:
   ```bash
   htdocs/
   ```
4. Open phpMyAdmin
5. Create required databases
6. Import SQL tables
7. Run project using:
   ```text
   http://localhost/
   ```
